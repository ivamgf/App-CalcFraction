Sobre o Aplicativo

Este aplicativo foi desenvolvido pelo LAMID para a disciplina de EGC - 510046 - Compartilhamento de Conhecimento e Acessibilidade Web. Trata-se de um simulador para operações com frações.


Tecnologia:
Esta aplicação foi desenvolvida com tecnologia Angular 9+, Nodejs, Sass, Typescript e Apache Cordova.
